# SteFin

A React dashboard demo built with Vite and Recharts.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Notes

- The app uses `SteFin.jsx` as the main component.
- Firebase config is stubbed in `SteFin.jsx`; demo mode works without it.
